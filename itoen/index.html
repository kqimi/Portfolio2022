ItoEn Website Re-Design

For my first project in interface design, I wanted to re-design a website for my favorite tea brand ItoEn. Through this project, I learned the importance of the design process by creating site maps, style tiles, wireframes, collecting images, and more. My target for this project was to create a website that is more suitable to the target audience and adding a new aspects that are trending for tea company.
